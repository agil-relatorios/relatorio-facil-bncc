Guia Rápido EducaNobre

1. Digite o nome do pequeno(a).

2. Selecione o campo da BNCC que deseja relatar.

3. Escolha o nível que mais combina com o momento atual da criança.

4. Clique em "Gerar" para criar o texto acolhedor.

5. Use o botão Copiar para levar o texto para o Word ou sistema da escola.
